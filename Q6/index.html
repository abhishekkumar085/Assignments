1. HTML
Primary Use Cases Key Features/Benefits
  • Building the Structure of Web Page
  • Creating Web Pages
  • SEO (Search Engine Optimization)
  • simple and easy to learn
  • compatible for all browser
  • Allow for Semantic markup
  1. CSS (Cascading Style Sheets)
Primary Use Cases Key Features/Benefits
  • Styling HTML Elements
  • Layout and Positioning
  • Responsive Web Design
  • Animations and Transitions
  • Selector-Based Styling
  • CSS provides various positioning properties, such as position, float, and display
  • Responsive web design (RWD) is an approach to web design and development that focuses on creating web pages that adapt and respond to various screen sizes and devices. It ensures that websites look and function well on desktop computers, laptops, tablets, and smartphones
  • Animations and transitions are important aspects of CSS (Cascading Style Sheets) that allow you to add motion and visual effects to web elements
  1. JavaScript
Primary Use Cases Key Features/Benefits
  • Enhancing Web Pages
  • DOM Manipulation
  • Real-Time Updates
  • JavaScript is commonly used to add interactivity and dynamic behavior to web pages. It allows you to respond to user actions, such as clicks and keystrokes, and update the content without requiring a full page reload.
  • JavaScript can manipulate the Document Object Model (DOM) of a web page, allowing developers to dynamically add, modify, or remove elements from the page.
  • JavaScript can be used to implement real-time features in web applications, such as live chat, notifications, and collaborative editing.
